# Ethiopian Flavors Memory Match 🇪🇹

A browser-based memory matching (concentration) game built as a **Web One** class project to practice **CSS** and **coordinate/layout techniques**. The game features pictures of traditional Ethiopian dishes, giving the project a fun cultural theme while focusing on core front-end skills.

## About the Project

This project was created to practice:
- CSS styling and positioning
- Grid/flexbox coordination and layout
- CSS transforms and transitions (card flip animations)
- DOM manipulation and event handling with JavaScript
- Basic game-state logic (matching pairs, tracking progress)

## How to Play

1. The board is filled with cards, each showing the back side (hidden) at the start.
2. Behind every card is a picture of an Ethiopian food dish. Each dish appears on exactly **two** cards, placed in random positions across the board.
3. Click a card to flip it and reveal the food picture underneath.
4. Click a second card to try to find its match:
   - If the two cards show the **same dish**, they stay face-up as a matched pair.
   - If they **don't match**, both cards flip back down after a short delay.
5. Continue selecting pairs of cards until **all matches are found**.
6. The game is complete when every pair on the board has been correctly matched!

## Features

- Randomized card shuffle on every game load/restart
- Smooth CSS flip animations
- Click-to-reveal card interaction
- Match detection logic
- Ethiopian cuisine theme (e.g., Doro Wat, Injera, Kitfo, Shiro, Tibs, and more)

## Technologies Used

- **HTML5** – structure and layout of the game board
- **CSS3** – styling, card flip animations, grid/flexbox coordination
- **JavaScript** – game logic, card matching, and interactivity

## Getting Started

1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Start matching!

No build tools or installations required — this is a pure HTML/CSS/JS project.
